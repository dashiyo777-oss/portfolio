# 学歴詐称・経歴詐称 評価ルール

## 1. 定義

| 種別 | 定義 |
|------|------|
| **学歴詐称** | 最終学歴・取得学位・資格・留学経験等の虚偽申告 |
| **経歴詐称** | 職歴・役職・在籍期間・実績の虚偽申告 |
| **経歴誇張** | 事実の誇大表現（詐称より軽度。「関与」→「主導」など） |

---

## 2. 重大度レベル

### Level A — 疑惑・調査中
- 報道機関・野党・第三者から指摘あり
- 本人が否定中または説明不十分
- 証拠未確定

### Level B — 認定済（軽微）
- 本人が事実を認めた、または証拠が確認済
- ただし刑事事件化・議員辞職には至っていない
- 誇張・不正確な表現レベル

### Level C — 認定済（重大）
- 重大な虚偽が確定（学位偽造・在籍証明偽造など）
- 選挙公報・公式プロフィールへの虚偽記載
- 辞職・公職選挙法違反等に発展

---

## 3. 軸スコアへの影響

### 政治倫理（第8軸）
| レベル | 上限値 | 備考 |
|--------|--------|------|
| Level A（疑惑） | ≤ 2 | 疑惑の間は上限2 |
| Level B（軽微認定） | ≤ 1 | 信頼毀損が確定 |
| Level C（重大認定） | 0 | 政治倫理の根幹に関わる |

### 説明責任（第5軸）
| レベル | 上限値 | 備考 |
|--------|--------|------|
| Level A | ≤ 2 | 説明不十分 |
| Level B | ≤ 2 | 認めたが対応が不十分 |
| Level C | ≤ 1 | 重大な虚偽の隠蔽 |

---

## 4. フラグ設定

| レベル | flag_caution | flag_crime | 備考 |
|--------|-------------|-----------|------|
| Level A | true | false | 要注意フラグのみ |
| Level B | true | false | 要注意フラグのみ |
| Level C | true | true | 両フラグを付与 |

---

## 5. evidence の記録方法

```json
{
  "cat": "問題・疑惑",
  "sub": "学歴詐称",
  "summary": "（一行要約）",
  "detail": "（詳細: 何をどのように詐称したか）",
  "src": "（報道元・公式資料など）",
  "url": "（URLがあれば）",
  "rel": "政治倫理",
  "impact": "高",
  "date": "（発覚・報道年月）"
}
```

**sub の例:**
- `"学歴詐称"` — 学位・学校の虚偽
- `"経歴詐称"` — 職歴・役職の虚偽
- `"経歴誇張"` — 誇大表現

---

## 6. 適用フロー（correction ファイル作成手順）

1. 報道ソースを確認し Level を判定
2. `政治倫理` 軸と `説明責任` 軸を上限規則に従って設定
3. total を再計算 (`int(sum(axes) × 100 / 40)`)
4. rank を再判定
5. `flag_caution: true`（Level C は `flag_crime: true` も）
6. evidence に `cat: "問題・疑惑"` エントリを追加（`_replace_evidence: false` — 既存実績は残す）
7. correction_XXX ファイルとして保存・マージ

---

## 7. 具体例

### 例1: 学歴誇張（Level A）
- 状況: 「〇〇大学卒業」と称していたが「中退」と報道
- 政治倫理: 2, 説明責任: 2
- flag_caution: true, flag_crime: false

### 例2: 学歴詐称認定（Level C）
- 状況: 存在しない大学の学位を公式プロフィールに記載
- 政治倫理: 0, 説明責任: 0
- flag_caution: true, flag_crime: true

### 例3: 職歴誇張（Level B）
- 状況: 「代表取締役」→実際は「平取締役」と本人認定
- 政治倫理: 1, 説明責任: 2
- flag_caution: true

---

## 8. 注意事項

- **疑惑段階（Level A）は reversible**: 疑惑が晴れた場合は flag_caution を外し、軸スコアを再評価
- **報道だけでは Level C 適用しない**: 証拠確定または本人認定が必要
- **政治倫理 0 は最重大**: flag_crime と同時にのみ使用
- **他の軸は中立評価を維持**: 詐称があっても政策能力軸（安全保障・経済等）は別途評価

## 9. 旧統一教会（世界平和統一家庭連合）関係議員の評価ルール

### 背景・根拠

- **2025年3月**: 東京地裁が解散命令（民法上の不法行為認定）
- **教団の反日的性格**: 「日本は贖罪のために財産を捧げるべき」という対日収奪的教義が記録されており、霊感商法等で国民から多大な被害をもたらした
- **2022年自民党調査**: 379人中179人が接点あり、121人の氏名公表

### 関係度 3 段階

| Level | 内容 | 政治倫理上限 | 説明責任上限 | フラグ |
|-------|------|------------|------------|--------|
| **L1 軽微** | 一度の会合出席・祝電のみ、その後断絶宣言済み | ≤ 3 | 制限なし | caution |
| **L2 中程度** | 継続的関係・資金受領・ボランティア支援受け入れ | ≤ 2 | ≤ 2 | caution |
| **L3 重大** | 組織的選挙動員受け入れ・高額資金授受・現在も関係継続 | ≤ 1 | ≤ 1 | caution |

### evidence の記録方法

```json
{
  "cat": "問題・疑惑",
  "sub": "旧統一教会関係",
  "summary": "（一行要約）",
  "detail": "（関係の具体的内容）",
  "src": "（報道元）",
  "url": "",
  "rel": "政治倫理",
  "impact": "高",
  "date": "（発覚・報道年月）"
}
```

### 注意事項

- **一度の出席だけでは L1 も適用しない**: 祝電・会合出席のみで後に完全断絶を宣言・実行している場合は適用対象外
- **政策軸は独立評価**: 関係の問題があっても安全保障・経済等の政策能力軸は別途評価
- **関係が断絶・解消された場合**: 時系列を evidence に明記し、継続中より軽い扱いとする

## 10. 外国勢力からの献金・資金受領の評価ルール

### 背景・根拠

- 政治資金規正法第22条の5: 外国人・外国法人からの政治献金を**明示的に禁止**
- 外国勢力からの資金が政策決定に影響を与えることは、国家主権・国民の参政権を侵害する
- **2006年改正の抜け穴**: 「5年以上上場の日本法人」であれば外資比率に関係なく献金可能（批判継続中）
- **2024年改正**でも抜け穴は温存

### 種別と重大度

| 種別 | 内容 | Level |
|------|------|-------|
| **種別A** 直接違法献金 | 外国人・外国法人からの直接献金（法律違反確定） | B〜C |
| **種別B** 外資系大量献金 | 外資比率50%超の法人から多額受領（合法だが倫理的問題・報道確認済み） | A〜B |

### 軸スコアへの影響

| Level | 政治倫理上限 | 説明責任上限 | フラグ |
|-------|------------|------------|--------|
| **Level A** 疑惑・調査中 | ≤ 2 | ≤ 3 | caution |
| **Level B** 法律違反確定・引責辞任等 | ≤ 2 | ≤ 3 | caution |
| **Level C** 刑事起訴・選挙違反等 | ≤ 1 | ≤ 2 | caution + crime |

※ 引責辞任して説明責任を果たした場合は 説明責任軸を Level B でも ≤ 3 に緩和

### evidence の記録方法

```json
{
  "cat": "問題・疑惑",
  "sub": "外国人違法献金",
  "summary": "（一行要約）",
  "detail": "（献金元・金額・対応の詳細）",
  "src": "（報道元）",
  "url": "",
  "rel": "政治倫理",
  "impact": "高",
  "date": "（発覚年月）"
}
```

**sub の例:**
- `"外国人違法献金"` — 外国人・外国法人からの直接献金（違法）
- `"外資系企業献金"` — 外資系企業からの多額献金（合法だが倫理的問題）

### 注意事項

- **少額・過失・即時返金済み**: 影響を軽減できる（-0.5 Level相当の扱い）
- **引責辞任・公表済み**: 説明責任軸の減点を緩和
- **合法の外資系企業献金**: 具体的な報道ソースによる金額・社名の特定が必要

## 11. 個人経歴の透明性（帰化・国籍歴の開示）

### 基本方針

- **帰化の事実をオープンに公表している議員**: ペナルティなし（透明性を果たしている）
- **帰化の事実を積極的に隠す・虚偽申告している議員**: ペナルティ対象
- **聞かれていないだけで言及していない**: 積極的隠蔽と認められなければ対象外

### 対象となる行為

帰化に限らず、**有権者の判断に関わる個人経歴を積極的に隠す・虚偽申告する**行為全般を対象とする。

- 帰化・国籍歴の隠蔽・虚偽申告
- 出生地・民族的背景の積極的虚偽申告
- 二重国籍の長期隠蔽（蓮舫の事例と同様）

### 軸スコアへの影響

既存の「学歴詐称・経歴詐称ルール」のサブカテゴリとして扱う。

| 確認状況 | 説明責任上限 | 政治倫理上限 | フラグ |
|----------|------------|------------|--------|
| 隠蔽の疑惑（報道あり） | ≤ 2 | ≤ 2 | caution |
| 積極的虚偽申告の確認 | ≤ 2 | ≤ 1 | caution |

### 注意事項

- **判明時のみ適用**: 確認できない段階では一切適用しない
- **報道ソースが必須**: 本人の過去発言・公的記録・報道等による確認が必要
- **帰化自体はペナルティゼロ**: オープンに公表している議員は影響なし
- **経歴詐称との統合**: 適用水準・evidence記録方法は Section 1〜8 に準拠
